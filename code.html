The Code

                    
function getValues() {
  let pitchValue = document.getElementById("pitchValue").value;
  let puttValue = document.getElementById("puttValue").value;
  let stopValue = document.getElementById("stopValue").value;

  let pitchNum = parseInt(pitchValue);
  let puttNum = parseInt(puttValue);
  let stopNum = parseInt(stopValue);

  if (
    Number.isInteger(pitchNum) &&
    Number.isInteger(puttNum) &&
    Number.isInteger(stopNum) &&
    stopNum > 0
  ) {
    let numberArray = generatePitchPutt(pitchNum, puttNum, stopNum);
    displayPitchPutt(numberArray);
  } else {
    Swal.fire({
      icon: "error",
      title: "Oops!",
      text: "Please enter positive integers for the pitch, putt, and stop values",
      backdrop: false,
    });
  }
}

function generatePitchPutt(pitch, putt, stop) {
  let resultArr = [];

  for (let number = 1; number <= stop; number++) {
    if (number % pitch == 0 && number % putt == 0) {
      resultArr.push("Pitch&Putt");
    } else if (number % putt == 0) {
      resultArr.push("Putt");
    } else if (number % pitch == 0) {
      resultArr.push("Pitch");
    } else {
      resultArr.push(number);
    }
  }

  return resultArr;
}

function displayPitchPutt(numbers, pitch, putt) {
  let results = "";

  for (let i = 0; i < numbers.length; i++) {
    let num = numbers[i];

    if (num % pitch === 0 && num % putt === 0) {
      results = results + `<tr><td>${"Pitch&Putt"}</td></tr>`;
    } else if (num % pitch === 0) {
      results = results + `<tr><td>${"Pitch"}</td></tr>`;
    } else if (num % putt === 0) {
      results = results + `<tr><td>${"Putt"}</td></tr>`; //<
    } else {
      results = results + `<tr><td>${num}</td></tr>`;
    }
  }

  let tableBody = document.getElementById("results");
  tableBody.innerHTML = results;
}

The code is structured into three separate functions. Below you can get a better understanding of how this code works.

getValues( )

This function retrieves the values of the pitchValue, puttValue, and stopValue elements from the HTML document. It then parses these values into integers using parseInt( ) and stores them in the variables pitchNum, puttNum, and stopNum. It performs some input validation checks using Number.isInteger( ) and ensures that all three values are positive integers. If the conditions are met, it calls the generatePitchPutt(pitchNum, puttNum, stopNum) function and passes the values as arguments. If the conditions are not met, it displays an error message using the Swal library.

generatePitchPutt(pitch, putt, stop)

This function takes three parameters: pitch, putt, and stop, which represent the input values for the pitch, putt, and stop. It initializes an empty array called resultArr to store the generated results. It then uses a for loop to iterate from 1 to stop, inclusive. For each number, it checks the conditions using modulo operations (%) to determine if the number is divisible by pitch, putt, or both. Depending on the conditions, it adds either "Pitch&Putt", "Putt", "Pitch", or the number itself to the resultArr. After the loop, it returns the resultArr with the generated results.

displayPitchPutt(numbers, pitch, putt)

This function takes three parameters: numbers, pitch, and putt. The numbers parameter is an array containing the generated results from the previous function. It initializes an empty string called results to store the HTML content. It then uses a for loop to iterate through each number in the numbers array. Inside the loop, it checks the conditions using modulo operations (%) to determine if the number should be classified as "Pitch&Putt", "Putt", "Pitch", or remain unchanged. It appends the appropriate HTML content to the results string based on the conditions. Finally, it retrieves the results element from the HTML document and sets its innerHTML to the results string, which displays the generated results in an HTML table format.
